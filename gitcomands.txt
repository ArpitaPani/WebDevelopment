# .gitignore file --> the files u don't want to share with anyone
git show --> gives us all the commit and changes made


git branch --> list all the branches present in the current repo
git status --> show the status of all the files in the current directory
git brach <branch_name> --> create a branch with the branch_name specified
git checkout <branch_name> --> change our branch
git checkout -b <branch_name> --> creates a new branch and moves us to that new branch
git push origin <branch_name> --> upload/update the commited files of ur local repo in ur online repo



git init --> initialize ur git repo/ local git repo
git add <filename> --> add the specific file to the staging area
git commit -m "Custom Message" --> commits/saves all the changes of those files which were staging area 
git add . --> it adds all the untracked and modified file to the staging area

git config --global user.name "<GitHub_username>" --> sets the author's username 
git config --global user.email "<GitHub_email_address>" --> sets the author's email
git cofig --list --> lists all the configuration of the repo/git
